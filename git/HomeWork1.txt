1. 
- Установить гит.
- Связать свой ПК с аккаунтом гита SSH ключом.
- Скопировать ссылку на репозиторий.
- Найти папку место у себя на ПК где будет вестилсь локальная разработка
- зарустить баш
- сделать клон репозитория git clone <ссылка с репозитория>.
2. 
- зарустить баш в локальной паке разработке
- Перейти на мастер ветку
- Сделать пул git pull.
3.
git status - покажет не отслеживаемые файлы
git log - покажет историю коммитов с их хэшом, по которому потом можно откатиться.
git remote -v - название удаленного репозитория.
